# 📊 Customer Churn Analysis (Power BI)

## 🚀 Overview

Analyzed customer churn using Power BI to identify key factors affecting customer retention and behavior.

---

## 🧰 Tools

* Power BI
* Excel / CSV
* Power Query

---

## 📊 Key Features

* Churn rate analysis
* Customer segmentation
* Tenure-based insights
* Interactive dashboard

---

## 🔍 Insights

* Higher churn in low-tenure customers
* Certain plans show higher churn
* Demographics impact retention

## 📚 Learnings

* Customer behavior analysis
